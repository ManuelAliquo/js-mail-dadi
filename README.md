# Mail

Crea una lista di email di invitati ad una festa.
Chiedi all’utente la sua email, controlla che sia nella lista e stampa un messaggio appropriato sull’esito del controllo utilizzando un ciclo for.

# Dadi

Generare un numero random da 1 a 6, sia per il giocatore sia per il computer.
Il numero più alto vince.
